About ShopFlask

Built with Python, Flask & a little ❤️

ShopFlask is a fully-featured ecommerce platform built as a learning project using Flask, SQLAlchemy, and uv for dependency management. It demonstrates a real-world application structure with authentication, shopping cart, checkout, order management, and Stripe payment integration hooks.

What's Included

User registration and login with secure password hashing. A product catalog with categories, search, sorting, and pagination. A fully functional shopping cart with quantity controls. A checkout flow with shipping address form. Order history and detail pages. A clean, responsive UI built with vanilla CSS — no external UI frameworks.

Tech Stack

Python 3.12+ — Flask 3.x — SQLAlchemy 2.x — Flask-Login — Flask-WTF — uv (package manager) — Stripe (payment integration stub).

Getting Started

Clone the repo, run uv sync to install dependencies, copy .env.example to .env, then run uv run flask run. The database is created and seeded automatically on first launch.
